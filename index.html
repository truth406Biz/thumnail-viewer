💡 상품 썸네일 뷰어 사용법

1. 데이터 준비하기

이미지 URL이 한 줄씩 나열된 데이터를 엑셀에서 CSV 파일 확장자(.csv)로 저장해 주세요. (http:// 또는 https:// 형식만 인식합니다.)

2. 파일 업로드

상단의 [파일 선택] 버튼을 눌러 저장한 CSV 파일을 불러오면 이미지가 화면에 나타납니다. 외부 서버로 전송되지 않으므로 대외비 데이터도 안전합니다.

3. ⌨️ 단축키 안내

  • Q 또는 ← : 이전 이미지
  • E 또는 → : 다음 이미지
  • 1 : 톡딜 썸네일 크기 (350px)
  • 2 : 톡딜 상품메인 크기 (450px)
  • 3 : 대화면 크기 (600px)

* 순서 입력창에 커서가 깜빡일 때는 단축키가 작동하지 않습니다. 입력 후 빈 화면을 클릭해 주세요!
